# Koa2 SocketIO

## Description

Chat App using :

* Koa 2
* SocketIO
* jQuery
* SCSS
* Asynchronous Functions (Async/Await)

## Running

Install dependencies

```
npm install
```

Start a Local Server

```
npm start
```

Run Test

```
npm test
```

Building and Running Production Server

```
npm run prod
```
